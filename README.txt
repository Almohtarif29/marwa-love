MARWA LOVE WEBSITE 💗🐱

How to use:
1) Open index.html in Chrome or any browser.
2) Click "شغّلي السحر والموسيقى" to start the music. Browsers do not allow music autoplay before a click.
3) To upload online, upload all files together exactly like this:
   - index.html
   - style.css
   - script.js
   - assets folder

Personalization:
- To change the day you started together, open script.js and change:
  const togetherSince = new Date('2026-06-18T00:00:00+01:00');

Music:
- I used the audio extracted from the first uploaded video and named it:
  assets/sata-9atala-stormy.m4a
- If you have the full SATA 9ATALA BY STORMY audio file, replace that file with your song using the same name.

Colors:
- The design uses pink and light blue because these are Marwa's favorite colors.

Made with love for Marwa.
